WHAT IS IT?
~~~~~~~~~~~

Azerty version (French Keyboard) for Caprice32 Raspberry Pi version (from cpc4rpi source  http://gaming.capsule-sa.co.za/downloads/cpc4rpi-1.1_src.tar.gz )
